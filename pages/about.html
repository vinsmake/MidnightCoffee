I'm an independent web developer, this web page is just a template created by me, inspired by an ideal coffee-about blog. So, in this section I'm going to show some resources I've used, like the colors and fonts.

About colors

  • #1B1B1B
  • #112229
  • #292929
  • #484848
  • #7B5F3F
  • #FFFFFF

First of all, I decided to use Adobe Color Wheel to select a gray-scale, also, I wanted to utilize one color that reminds the midnight's blue and coffee's tone.

To be sure that the color palette would not have accessibility problems, I decided to use the Google Color Tool which showed me that with the min 48% opacity, the #FFFFFF font color would not have problems

Fonts

I wanted a font that simulates the newspaper headlines, that's why I chose The Newspaper Font as my titles font, that font's serif helps to gide the user's eyes and makes easy to understand that the titles with serif are exactly what they are, titles.

Midnight Coffee's Titles

But, the serif fonts are not exactly easy to read, there's when a second sans serif font takes place. My chose for an easier reading experience was the Source Sans Pro, a sans serif font. This makes texts more easy-to-read.

This is an example of a Sans Serif font.

Finally, it's important to think about the Gestalt Principle of similarity and proximity, so, for the blog titles and call-to-action buttons I used the same sans serif font, but with bigger bold and font size.
